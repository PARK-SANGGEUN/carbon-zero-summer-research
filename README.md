# 모두의 탄소제로 탐구생활

춘천시 탄소중립지원센터 캠페인 포스터에서 출발한 고등학교 교과 융합 탐구 활동 웹페이지입니다.
포스터 속 5가지 탄소중립 실천(적정온도 유지, 차광, 다회용기 사용, 제철음식, 대기전력 차단)을
2015·2022 개정 교육과정 과목·단원과 매칭하고, 난이도 하·중·상의 탐구 활동과
인터랙티브 가설 검증 시뮬레이터(즉시 체험형 / 변인통제 실험 / 대조군·실험군 비교)를 제공합니다.

## 보는 방법
1. 이 저장소를 GitHub Pages로 배포하거나
2. `index.html`을 `poster.png`와 같은 폴더에 두고 브라우저에서 직접 엽니다.

## GitHub Pages 배포 방법
1. Settings → Pages → Source를 `main` 브랜치, `/ (root)` 폴더로 설정
2. 저장 후 `https://[사용자명].github.io/[저장소명]/` 주소로 접속

## 파일 구성
- `index.html` — 메인 페이지 (인트로 애니메이션, 포스터, 교과 매칭, 난이도별 시뮬레이터)
- `poster.png` — 춘천시 탄소중립지원센터 캠페인 포스터 원본 이미지
